\section*{Wielomiany Legendre'a}

Wzór ogólny:
\begin{itemize}
    \item $L_0(x) = 1$
    \item $L_n(x) = \frac{1}{2^n\cdot n!}\cdot\frac{d^n}{dx^n}
    \left(\left(x^2-1\right)^n\right)$
\end{itemize}

Wzór rekurencyjny:
\begin{itemize}
    \item $L_0(x) = 1$
    \item $L_1(x) = x$
    \item $L_{n+1}(x) = \frac{2n+1}{n+1} x \cdot
    L_n(x) - \frac{n}{n+1}\cdot L_{n-1}(x)$
\end{itemize}
